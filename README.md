## 📁 项目列表

### 1. SaaS 产品落地页 — “智析”数据协作平台官网

- **在线演示**：[点击查看](https://Kiri060127.github.io/portfolio-projects/dashboard.html)
- **项目定位**：还原售前为客户展示方案效果的场景，搭建可交互的数据演示 Demo
